Package Metadata
================

Module: :mod:`thornforge`

Overview
--------

The package module exposes metadata such as the project title, summary,
homepage, version, author, and email address. Installed distributions read this
from package metadata. Source checkouts fall back to ``pyproject.toml`` so local
tests and docs builds can run before installation.
