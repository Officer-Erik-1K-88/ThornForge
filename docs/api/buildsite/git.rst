Git And Hashing Helpers
=======================

Module: :mod:`thornforge.buildsite.git`

Overview
--------

This module wraps Git subprocess calls, exports tagged refs, clones remote
repositories, copies local worktrees, and computes the content digests used to
deduplicate documentation builds.
