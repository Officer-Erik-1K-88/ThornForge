Info Site Publishing
====================

Module: :mod:`thornforge.buildsite.info_site`

Overview
--------

The info-site helpers publish an optional ``info/`` subtree as the root portion
of the generated static site. Text-like files are rendered into wrapped HTML,
HTML files are wrapped with shared navigation, and assets are copied through.
