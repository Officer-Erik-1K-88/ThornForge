API Reference
=============

ThornForge's public surface is currently the command line and the
``build_versioned_site`` function. The lower-level modules are useful for
contributors and tests, but are not yet a stable third-party API.

The API reference is organized in layers:

* top-level package and CLI pages
* buildsite package landing pages
* focused module pages for repository discovery, Git helpers, site assembly, and
  HTML rewriting

Use the pages below to drill into the part of the package you want.
